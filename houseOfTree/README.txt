Ακολουθήστε τα εξής βήματα για δημιουργικά του αρχείου στο Netbeans:
1. Στο IDE δημιουργείστε νέο project με όνομα FamilyTree μαζί και την Main class
2. Στον φάκελο του αρχείου κάνουμε επικόλληση το csv
3. Αντικαθιστούμε το αρχείο java FamilyTree το μεταφέρουμε στο src
4. Στον φάκελο του αρχείου FamilyTree.java κάνουμε επικόλληση τον φάκελο classes
5. Διαγράφουμε τον φάκελο FamilyTree στο src
6. Κλείστε το IDE και ανοίξτε τον ξανά
7. Επιλέγουμε το FamilyTree.java και όταν το τρέξουμε επιλέγουμε Οκ

Ακολουθήστε τα εξής βήματα για δημιουργικά του αρχείου στο intellij:
1. Στο IDE δημιουργείστε νέο project με όνομα FamilyTree
2. Κάντε Refactor rename την public class σε FamilyTree
3. Δημιουργήστε package με όνομα classes στο package που βρίσκεται η FamilyTree.java
4. Δημιουργήστε ένα αρχείο Java class με όνομα Person
5. Δημιουργήστε ένα αρχείο Java class με όνομα Tree
6. Κλείστε το IDE και ανοίξτε τον φάκελο που βρίσκεται το project
7. Διαγράψτε το περιεχόμενο του src και αντικαταστήστε το με της εργασίας
8. Στον φάκελο του αρχείου κάνουμε επικόλληση το csv
9. Ανοίξτε το IDE και τρέξτε το πρόγραμμα
